# Design System\_\_\_

**UI/UX standards and component library documentation.**

## Documents

### [Complete Guide\_\_](./design-system)

Full design system standards including atomic design, layouts, and variants.

**Audience**: Designers, Frontend developers

### [Component Catalog](./components/component-catalog)

**Documentation for every component built in Delta Labs. When you add a component to the app, add its doc here (props, variants, examples).**

**Audience**: Designers, Frontend developers  
**Focus**: Per-component docs — Button, Modal, DataTable, etc.

### [Design Tokens](./design-tokens)

Colors, typography, spacing, shadows, and foundational tokens. Use tokens instead of hardcoded values.

**Audience**: Designers, Frontend developers  
**Focus**: Token catalog and usage

### [API Standards](./api-standards)

REST conventions, authentication, DataContext patterns, and rate limiting for backend/frontend integration.

**Audience**: Backend and frontend developers  
**Focus**: API contract and integration
